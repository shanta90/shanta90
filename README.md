## 🚀 Projects

### 1. Job Application Tracker
Job Application Tracker is a small static web app built with HTML, vanilla JavaScript, Tailwind CSS (CDN), and DaisyUI. It lists jobs and lets you track status with dashboard counts (Total, Interview, Rejected), filter tabs, and per-job actions (move to Interview/Rejected, delete). Data is kept in memory in the browser, so refreshing the page resets the list.

🔗 **Live Demo:** https://shanta90.github.io/assignment-04  
💻 **Repository:** https://github.com/shanta90/assignment-04  

---

### 2. DigiTools Platform
DigiTools Platform is a modern, responsive digital tools marketplace built with React 19 and Vite. Users can browse premium digital products (AI tools, design templates, stock assets, and more), add items to a cart, and use a polished shopping-style flow. The UI uses a purple-focused gradient design, smooth interactions, and a layout that works on mobile and desktop. Styling uses Tailwind CSS and DaisyUI; notifications use react-toastify; icons use lucide-react; product data comes from JSON.

🔗 **Live Demo:** https://aquamarine-cassata-e283c5.netlify.app/  
💻 **Repository:** https://github.com/shanta90/Assignment-06

### 3. GitHub Issue Tracker
A responsive GitHub-style issue tracker with demo login, API-backed issue listing, status tabs (All / Open / Closed), search, stats, and issue details in a modal—built with HTML, Tailwind CSS (CDN), Font Awesome, and vanilla JavaScript.

🔗 **Live Demo:** https://shanta90.github.io/assignment-5/
